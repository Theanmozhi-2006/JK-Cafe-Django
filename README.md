# E-Commerce Project (Food Delivery Website)

A full-stack food delivery web application developed using Django. The platform allows users to browse food items, manage their cart, place orders, and interact with a responsive user interface.

## Features

* User registration and login
* Browse food items and categories
* Add and remove items from cart
* Place food orders
* Order management
* Responsive user interface
* Database integration for storing user and order information

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript

## Learning Outcomes

This project helped me gain practical experience in:

* Backend development with Django
* Database design and integration
* User authentication and session management
* Frontend development using HTML, CSS, and JavaScript
* Building full-stack web applications

## How to Run

1. Apply migrations:
   python manage.py migrate

2. Start the development server:
   python manage.py runserver

3. Open the application in your browser:
   http://127.0.0.1:8000/
